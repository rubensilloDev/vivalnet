## Core Web Vitals Recommendations

### Performance Issues:

| Metric | Risk | Cause | Fix |
|--------|------|-------|-----|
| **LCP** | ⚠️ Medium | Large hero images, external resources | Preload critical images, use WebP |
| **CLS** | ⚠️ Medium | Images without dimensions | Add width/height attributes |
| **FID** | ✅ Low | Minimal JavaScript | Keep scripts optimized |
| **TTFB** | ⚠️ Medium | Local development | Will improve with hosting |

### Quick Performance Wins:
1. **Preload hero images:**
   ```html
   <link rel="preload" as="image" href="./img/img-hero.jpeg" fetchpriority="high">
   ```

2. **Add image dimensions:**
   ```html
   <img src="..." width="800" height="600" alt="...">
   ```

3. **Lazy load below-fold:**
   ```html
   <img src="..." loading="lazy" alt="...">
   ```

4. **Optimize external images:**
   - Download Unsplash images
   - Convert to WebP format
   - Compress with tools like TinyPNG
